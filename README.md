# Tuning Show

Landing page for Tuning Show — Armenia's car culture event, September 20, 2026 at Garage Mall, Yerevan.

Built with React + Vite. Multilingual (Armenian / English / Russian), live countdown, 11-nomination showcase, participant/partner registration forms.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploys automatically to GitHub Pages on push to `main` via GitHub Actions.
